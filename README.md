# 📚 StudyLoom

**StudyLoom** is a simple Pinterest-inspired digital study planner designed to help students organize their academic life in a beautiful, cozy, and productive environment.

Unlike traditional productivity dashboards, StudyLoom focuses on aesthetics and user experience, combining the visual appeal of Pinterest, the organization of Notion, and the comfort of digital journals.

## ✨ Features

### ✅ Task Management

* Add tasks
* Edit tasks
* Delete tasks
* Mark tasks as completed
* Priority levels (High, Medium, Low)

### 📖 Subject Management

* Create custom subjects
* Add topics inside subjects
* Automatically calculate subject completion

### 📝 Notes Wall

* Create notes
* Edit notes
* Delete notes
* Sticky-note inspired interface

### 📊 Progress Analytics

* Overall task completion percentage

## 🎨 Design Inspiration

StudyLoom is inspired by:

* Pinterest
